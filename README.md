# Desarrollo de Software para Plataformas Móviles

Repositorio académico correspondiente a la asignatura **Desarrollo de Software para Plataformas Móviles** de la Universidad Autónoma de Occidente.

## Descripción de la Asignatura

Esta materia está orientada al diseño, desarrollo e implementación de aplicaciones móviles híbridas utilizando tecnologías modernas para la creación de soluciones multiplataforma.

Durante el curso se abordaron conceptos fundamentales y avanzados relacionados con el desarrollo móvil, integrando herramientas y tecnologías como **React**, **Ionic Framework**, **TypeScript**, **Capacitor** y **Firebase** para la construcción de aplicaciones modernas orientadas a dispositivos Android y entornos web.

El enfoque de la asignatura combina teoría y práctica, permitiendo aplicar conocimientos de arquitectura de software, diseño de interfaces móviles, integración de sensores del dispositivo, consumo de servicios web y almacenamiento de datos en la nube.
